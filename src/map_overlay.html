Albuquerque Map Maker - Overlays

Visualizing useful data for social and environmental justice work in Albuquerque, New Mexico

How to use this map

This map is meant to be a data aggregator for Albuquerque.
Click and drag in the map to pan your view. Use the "+" and "-" buttons to zoom the view in or out.
To add data layers or change the basemap click on the layers icon in the upper right of the map, which opens a popup menu.
On the popup menu you can select the layers you want to add to the map or use the radio buttons to change the basemap.
Click the icon shaped like a downward arrow to download a .png file of the map you have created.
